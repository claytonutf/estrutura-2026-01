O que é uma fila encadeada?

Uma fila encadeada é uma estrutura de dados linear em que os elementos são alocados dinamicamente na memória heap e conectados por ponteiros. Ao contrário do vetor, não há limite fixo de tamanho.

↓ ENQUEUE (inserção no final)
INÍCIO
A
B

↑ DEQUEUE (remoção no início)

Princípio FIFO

  • Quem entra primeiro, sai primeiro
  • Inserção sempre no final da fila
  • Remoção sempre no início da fila
  • Encadeamento via ponteiro *prox
  • Memória alocada dinamicamente com new
🔗

Encadeamento Dinâmico

Cada nó aponta para o próximo via ponteiro *prox. O último nó aponta para nullptr, indicando o fim.

⚡

Sem Limite de Tamanho

Ao contrário de um array estático, a fila encadeada cresce dinamicamente conforme a necessidade, limitada apenas pela memória disponível.

📌

Dois Ponteiros de Controle

A struct fila mantém ponteiros para o início (remoção) e para o final (inserção).

🎯

Gerenciamento de Memória

Cada inserção aloca um nó com new. Cada remoção libera a memória com delete, evitando vazamentos.

Funções Implementadas

Cada função tem responsabilidade única e complexidade definida.

insere_fila(fi, al)

Enqueue

Aloca novo nó com new, define prox = nullptr, atualiza o ponteiro final. Se a fila estava vazia, atualiza também inicio.

Complexidade:O(1)
remove_fila(fi)

Dequeue

Guarda ponteiro para o primeiro nó, avança inicio para o próximo. Se a fila ficou vazia, atualiza final = nullptr. Libera memória com delete.

Complexidade:O(1)
consulta_fila(fi, *al)

Front / Peek

Copia os dados do nó no início da fila para o ponteiro fornecido. Não remove o elemento. Retorna 0 se a fila estiver vazia.

Complexidade:O(1)
tamanho_fila(fi)

Size

Percorre todos os nós a partir de inicio, contando elementos. Não há contador interno na struct, portanto a travessia é necessária.

Complexidade:O(n)
libera_fila(fi)

Free / Destructor

Loop enquanto inicio != nullptr: guarda nó atual, avança início, chama delete. Ao final, libera a própria struct fila.

Complexidade:O(n)
cria_fila()

Create

Aloca a struct fila no heap. Inicializa ambos os ponteiros (inicio e final) com nullptr, representando fila vazia.

Complexidade:O(1)

// Insere elemento no FINAL da fila — O(1)
int insere_fila(Fila fi, aluno al) {

    if (fi == nullptr) return 0;

    // aloca novo nó no heap
    elemento *no = new elemento;
    if (no == nullptr) return 0;

    no->dados = al;        // copia dados
    no->prox  = nullptr; // será o último

    if (fi->inicio == nullptr) {
        // fila vazia: início e final apontam para o único nó
        fi->inicio = no;
        fi->final  = no;
    } else {
        // fila não vazia: encadeia no fim
        fi->final->prox = no; // último aponta para novo
        fi->final       = no; // atualiza ponteiro final
    }
    return 1;
}
// Remove elemento do INÍCIO da fila — O(1)
int remove_fila(Fila fi) {

    if (fi == nullptr || fi->inicio == nullptr)
        return 0;

    // guarda referência ao primeiro nó
    elemento *no = fi->inicio;

    // avança o início para o próximo
    fi->inicio = fi->inicio->prox;

    // se ficou vazia, final também é nullptr
    if (fi->inicio == nullptr)
        fi->final = nullptr;

    // libera memória do nó removido
    delete no;
    return 1;
}
// Consulta (peek) o primeiro elemento — O(1)
int consulta_fila(Fila fi, aluno *al) {

    if (fi == nullptr || fi->inicio == nullptr)
        return 0;

    // copia dados do primeiro nó para *al
    *al = fi->inicio->dados;

    // NÃO remove — apenas lê
    return 1;
}
// Libera toda a memória da fila — O(n)
void libera_fila(Fila fi) {

    if (fi == nullptr) return;

    elemento *no;

    // percorre e desaloca nó por nó
    while (fi->inicio != nullptr) {
        no = fi->inicio;
        fi->inicio = fi->inicio->prox;
        delete no; // libera nó atual
    }

    // libera a própria struct fila
    delete fi;
}
int main() {
    // dados de exemplo
    aluno a[4] = {
        {1, "Andre",   9.5, 7.8, 8.5},
        {2, "Ricardo", 7.5, 8.7, 6.8},
        {3, "Bianca",  9.7, 6.7, 8.4},
        {4, "Ana",     5.7, 6.1, 7.4}
    };

    Fila fi = cria_fila(); // aloca fila vazia

    // insere 4 alunos na fila
    for (int i = 0; i < 4; i++)
        insere_fila(fi, a[i]);

    // consulta quem está no início
    aluno aux;
    if (consulta_fila(fi, &aux))
        cout << "Primeiro: " << aux.nome;

    // remove todos — ordem FIFO
    for (int i = 0; i < 4; i++)
        remove_fila(fi);

    libera_fila(fi); // sem vazamento!
    return 0;
}
