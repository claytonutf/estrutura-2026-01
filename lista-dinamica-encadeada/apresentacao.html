// 01 — Conceito

O que é uma Lista Encadeada?

Ao contrário de um array, os elementos não ficam em posições contíguas de memória. Cada elemento aponta para o próximo, formando uma cadeia de ponteiros.

✦ Tamanho dinâmico

Cresce e encolhe em tempo de execução com new e delete. Sem desperdício de memória.

⇄ Inserção eficiente

Inserir no início ou no meio é O(1) — basta redirecionar ponteiros. Não precisa deslocar elementos.

↗ Acesso sequencial

Para chegar ao nó N é preciso percorrer N-1 nós. Não há acesso aleatório por índice.

⚙ Gerência manual

O programador controla cada alocação (new) e desalocação (delete). Responsabilidade total.

// 02 — Estrutura

Anatomia do Nó

O código define duas structs e um typedef que trabalham juntos.

structs.cpp 
// Payload — os dados do aluno
struct aluno{
    int   matricula;   // chave
    char  nome[30];   // nome
    float n1, n2, n3; // notas
};

// Nó da lista
struct elemento{
    aluno     dados; // info
    elemento *prox;  // →
};

// Alias conveniente
typedef elemento* Lista;
Por que Lista é um ponteiro?
typedef elemento* Lista define que Lista é um ponteiro para o primeiro nó. Quando a lista está vazia, ele vale nullptr.
Ponteiro para ponteiro (Lista*)
As funções recebem Lista* (ponteiro para ponteiro). Isso permite alterar o endereço do primeiro nó dentro da função — essencial para inserções no início.
char nome[30]
O nome é um array de char de tamanho fixo — herança da linguagem C. Para strings flexíveis use std::string em C++ moderno.
// 03 — Operações

As 5 Funções da Lista

Cada operação manipula ponteiros de forma cirúrgica.

cria_lista()

Aloca na heap um ponteiro para Lista e inicializa com nullptr.

cria_lista() 
Lista* cria_lista(){
    Lista* li = new Lista; // aloca ponteiro na heap
    if(li != nullptr)
        *li = nullptr;       // lista começa vazia
    return li;
}

insere_lista_ordenada()

Percorre a lista comparando matrículas até achar a posição correta, depois redireciona os ponteiros.

1

Aloca novo nó

elemento *no = new elemento — reserva memória na heap para o nó e copia os dados do aluno.

2

Avança até a posição correta

Loop while compara atual->dados.matricula < al.matricula e avança ant e atual enquanto a posição correta não é encontrada.

3

Inserção no início (ant == nullptr)

O novo nó passa a ser o primeiro: no->prox = *li; *li = no.

4

Inserção no meio / fim

no->prox = atual; ant->prox = no — o nó anterior passa a apontar para o novo, que por sua vez aponta para o restante da lista.

insere_lista_ordenada() 
int insere_lista_ordenada(Lista* li, aluno al){
    if(li == nullptr) return 0;

    elemento *no = new elemento;
    if(no == nullptr) return 0;

    no->dados = al; // copia dados

    elemento *ant = nullptr;
    elemento *atual = *li;

    // avança até posição correta
    while(atual != nullptr &&
          atual->dados.matricula < al.matricula){
        ant = atual;
        atual = atual->prox;
    }

    if(ant == nullptr){ // inserção no início
        no->prox = *li;
        *li = no;
    } else {           // meio ou fim
        no->prox = atual;
        ant->prox = no;
    }
    return 1;
}

remove_lista_final()

Percorre até o último nó, desconecta o penúltimo e libera a memória.

remove_lista_final() 
int remove_lista_final(Lista* li){
    if(li == nullptr || *li == nullptr)
        return 0;

    elemento *ant = nullptr;
    elemento *no  = *li;

    // avança até o último nó
    while(no->prox != nullptr){
        ant = no;
        no = no->prox;
    }

    if(ant == nullptr)  // só 1 elemento
        *li = nullptr;
    else
        ant->prox = nullptr; // desconecta

    delete no; // libera memória
    return 1;
}

libera_lista()

Percorre todos os nós e libera a memória de cada um — evita memory leak.

Memory leak acontece quando você delete o ponteiro da lista mas esquece de liberar cada nó individualmente. Cada new deve ter exatamente um delete correspondente.
libera_lista() 
void libera_lista(Lista* li){
    if(li != nullptr){
        elemento* no;
        while(*li != nullptr){
            no = *li;              // guarda referência
            *li = (*li)->prox;     // avança
            delete no;             // libera nó
        }
        delete li;             // libera ponteiro da lista
    }
}
// 04 — Memória

Stack vs Heap

A lista vive na heap. O ponteiro inicial (li) fica na stack.

📥 Stack

Variáveis locais (li, i, no, ant, atual). Gerenciadas automaticamente — surgem na chamada da função e somem no retorno.

🔧 Heap

Os nós alocados com new. Ficam vivos até você chamar delete. O ponteiro da lista (*li) também está aqui.

⚠ Memory Leak

Se o programa terminar sem chamar libera_lista(), todos os nós ficam órfãos na heap — memória vazada que o SO terá de recuperar.

// 05 — Complexidade

Análise de Desempenho

Cada operação tem um custo. Conhecê-lo guia a escolha da estrutura certa.

Operação Complexidade Motivo
cria_lista() O(1) Apenas uma alocação e uma atribuição.
insere_lista_ordenada() O(n) Percorre até achar a posição certa — até n nós no pior caso.
remove_lista_final() O(n) Precisa chegar ao último nó — percorre toda a lista.
tamanho_lista() O(n) Conta nó por nó sem cache de tamanho.
imprime_lista() O(n) Visita todos os n nós para imprimir.
libera_lista() O(n) Desaloca cada um dos n nós individualmente.
Inserção no início (variação) O(1) Não é ordenada, mas possível com a mesma estrutura.
Dica de otimização: Armazenar o tamanho em uma variável dentro da estrutura reduz tamanho_lista() de O(n) para O(1). O mesmo vale para um ponteiro ao último nó, que tornaria remove_lista_final() O(1) com uma lista duplamente encadeada.

Código-fonte Anotado

Todos os blocos reunidos e comentados.

lista_encadeada.cpp 
#include <iostream>
using namespace std;

/* ── DADOS ─────────────────────────── */
struct aluno{
    int   matricula;
    char  nome[30];
    float n1, n2, n3;
};

/* ── NÓ DA LISTA ───────────────────── */
struct elemento{
    aluno     dados;
    elemento *prox;
};

typedef elemento* Lista;

/* ── CRIA ──────────────────────────── */
Lista* cria_lista(){
    Lista* li = new Lista;
    if(li != nullptr) *li = nullptr;
    return li;
}

/* ── LIBERA ────────────────────────── */
void libera_lista(Lista* li){
    if(li != nullptr){
        elemento* no;
        while(*li != nullptr){
            no = *li;
            *li = (*li)->prox;
            delete no;
        }
        delete li;
    }
}

/* ── TAMANHO ───────────────────────── */
int tamanho_lista(Lista* li){
    if(li == nullptr) return 0;
    int cont = 0;
    elemento* no = *li;
    while(no != nullptr){ cont++; no = no->prox; }
    return cont;
}

/* ── INSERÇÃO ORDENADA ─────────────── */
int insere_lista_ordenada(Lista* li, aluno al){
    if(li == nullptr) return 0;
    elemento *no = new elemento;
    if(no == nullptr) return 0;
    no->dados = al;
    elemento *ant = nullptr, *atual = *li;
    while(atual != nullptr &&
          atual->dados.matricula < al.matricula){
        ant = atual; atual = atual->prox;
    }
    if(ant == nullptr){ no->prox = *li; *li = no; }
    else { no->prox = atual; ant->prox = no; }
    return 1;
}

/* ── REMOÇÃO FINAL ─────────────────── */
int remove_lista_final(Lista* li){
    if(li == nullptr || *li == nullptr) return 0;
    elemento *ant = nullptr, *no = *li;
    while(no->prox != nullptr){ ant = no; no = no->prox; }
    if(ant == nullptr) *li = nullptr;
    else ant->prox = nullptr;
    delete no;
    return 1;
}

/* ── IMPRESSÃO ─────────────────────── */
void imprime_lista(Lista* li){
    if(li == nullptr) return;
    elemento* no = *li;
    while(no != nullptr){
        cout << "Matrícula: " << no->dados.matricula << endl;
        cout << "Nome: "      << no->dados.nome      << endl;
        cout << "Notas: "
             << no->dados.n1 << " "
             << no->dados.n2 << " "
             << no->dados.n3 << endl;
        no = no->prox;
    }
}

/* ── MAIN ──────────────────────────── */
int main(){
    aluno a[4] = {
        {2,"Andre",  9.5,7.8,8.5},
        {4,"Ricardo",7.5,8.7,6.8},
        {1,"Bianca", 9.7,6.7,8.4},
        {3,"Ana",    5.7,6.1,7.4}
    };
    Lista* li = cria_lista();
    for(int i = 0; i < 4; i++)
        insere_lista_ordenada(li, a[i]);
    imprime_lista(li);
    for(int i = 0; i < 4; i++)
        remove_lista_final(li);
    for(int i = 0; i < 4; i++)
        insere_lista_ordenada(li, a[i]);
    libera_lista(li);
    return 0;
}
